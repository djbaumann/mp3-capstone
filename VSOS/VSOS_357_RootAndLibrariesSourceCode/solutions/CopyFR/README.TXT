
        CopyF, CopyR, Cmp, MkDir v1.03
        ------------------------------
           for VSOS 3.50 or higher
          2017-12-05  VLSI Solution




CopyF and CopyR are VSOS Shell programs that copy a file, or a
recursive folder structure, respectively. Cmp compares two files.
MkDir creates a directory. Run programs with the -h option for
more information.
