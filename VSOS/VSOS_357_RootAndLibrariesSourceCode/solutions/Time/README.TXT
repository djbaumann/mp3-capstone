
                TIME 1.01
                ---------
          for VSOS 3.50 or higher
         2017-12-05  VLSI Solution




Time is a VSOS Shell program that displays the time spent to run a program,
and its error code.




Usage: time [program [parameters]|-h]
-h      Show this help




Example:
S:>time setclock -t
CPU speed: effective speed 79.08/86.02 MHz = 91.9%, overhead 6.94 MHz
errCode 0, time:  1.093s
